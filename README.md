# mockxhr ![build status](https://travis-ci.org/wizawu/mockxhr.svg)

Flyweight ajax and mock-ajax built on Vanilla JS.

### Installation

#### Node

```
npm install --save mockxhr
```

#### Browser

```
<script src="dist/mockxhr.min.js"></script>
```
